Our model, XSS Threat Detection (Using AI), is designed to identify whether a given script has the potential to exploit a vulnerability known as Cross-Site Scripting (XSS).

What is XSS?

Cross-Site Scripting is a type of attack where malicious scripts are injected into trusted websites. These scripts can steal sensitive information, manipulate web content, or even take control of a userâ€™s session.

Example Scenario

1. Normal Input: A user submits their name in a form:

    John Doe

The website safely displays:

    Welcome, John Doe!

2. Malicious Input: An attacker submits a script instead of their name:

    <script>alert('XSS Attack!');</script>

If the website is not secure, it will execute the script, displaying an alert box with the message "XSS Attack!".

How our model helps?

Our model scans inputs like the one above and predicts if they are:

Safe: 

Hello, world!

Prediction: Not XSS

Potentially Malicious:

<img src="x" onerror="alert('XSS!')">

Prediction: Potential XSS
